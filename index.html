Web

Django

This framework is predominant in my web development experience. Here are the projects I created based on it:

Q-Writer - I originally created this project to display records of my Telegram bot, a bot that complemented the text sent by the user. All these additions could be viewed using the site in tape format. I later rewrote this site for the news.

Telegram

aiogram

In addition to web development, I also created bots for Telegram Messenger. It has a very flexible API and is also free. Therefore, it is very interesting to use and explore it. Here are the projects I created with aiogram:

Coiner - primitive economic game in it you need to raise money by being active in groups where there is this bot, for which the bot donates money. You can also try to get a valuable item through roulette, or buy yourself slaves who will do everything for you.

File hostinger - this bot uses the Telegram BOT API feature, the messenger allows you to download files via HTTP from Telegram servers, but weighing no more than 20 megabytes, this is enough to place a photo or a small video. The bot acts as a manager between the Telegram and the user. All you have to do is upload the file to the chat with the bot, then the bot will perform all the actions from the API, and provide a link where you can view the media file (not just media).

Imager - this project was mentioned in the section with Django. The point of the bot is that users can add their media to a shared collection, while others can view the collection in order or randomly. The bot independently sorts the media by type, the user only needs to send them and everything. At the moment, the development of the bot continues.
